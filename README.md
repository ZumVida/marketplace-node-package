# marketplace-service

## Install

Setup registry

```shell
npm config set registry https://nodejs-registry.zumvida.com
```

```shell
# NPM
npm i @zumvida/marketplace-service

# YARN
yarn add @zumvida/marketplace-service

# PNPM
pnpm add @zumvida/marketplace-service
```
